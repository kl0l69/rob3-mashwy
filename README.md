## Overview

**Rub3 Mashwy Website** is a fully responsive, client-side restaurant website designed to:

- Showcase a complete menu of grilled meats, vegetables, soups, salads, desserts, and beverages.
- Allow users to place orders directly via WhatsApp.
- Provide a modern, interactive, and visually appealing layout.

> **Note:** This website works entirely on the client side; no backend is required.

---

## Key Features

- Responsive homepage with a banner: "الطعم المشوي الأصلي – طعم مصري أصيل".
- Complete menu with categories: المشويات، الشوربة، السلطات، الوجبات، المحاشي، الحلويات، المشروبات.
- Each menu item includes:
  - Name
  - Price
  - Quantity input
  - Optional notes
  - WhatsApp order button
- Contact section for developer with social links
- Reusable React components: Header, Footer, MenuItem
- TailwindCSS styling for modern, warm restaurant look

## Quick Start

### Requirements

- Node.js 18+  
- npm or yarn  
- Git  

### Installation

```bash
git clone https://github.com/kl0l69/rub3-mashwy.git
cd ruba3-mashwy
npm install
npm start
````

### WhatsApp Ordering

Clicking "اطلب على واتساب" for any menu item opens WhatsApp with a pre-filled message containing:

```
مرحبا، أريد طلب [quantity] من [item name]
ملاحظات: [optional notes]
```
